11. Aplikasi yang dipakal Tindakan Percobaan Aplikasi pengolah lembar kerja Bukalah sebuah file .docx yang telah disiapkan guru. Simpan sebuah file .docx menjadi .rtf dan buka file .rtf tersebut. Aplikasi pengolah Simpan sebuah file.docx menjadi kata pdf dan buka file .pdf tersebut. Simpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut. Buka sebuah file lembar kerja (.xlsx). Simpan sebuah file .xlsx menjadi .txt dan buka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .csv dan buka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .pdf dan buka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .docx Apa yang terjadi? Apa kesimpulan kalian? Mengapa demikian?​


Percobaan dengan word processor atau pengolah kata:

Jika kita menyimpan sebuah file .docx menjadi .rtf dan membuka file .rtf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .rtf, yang merupakan rich text format.  Jika kita menyimpan sebuah file .docx menjadi .pdf dan membuka file .pdf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .pdf, yang merupakan format dokumen portabel (portable document format). Jika kita menyimpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut, yang terjadi adalah file .docx akan disimpan dengan ekstensi .xlsx, tetapi tidak akan dikonversi ke format .xlsx, yang merupakan format spreadsheet (lembar kerja).

Pembahasan

Ada pun bila dilakukan percobaan dengan aplikasi spreadsheet atau pengolah lembar kerja:

Jika kita membuka sebuah file lembar kerja (.xlsx) dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan terbuka dengan normal, karena formatnya sesuai. Jika kita menyimpan sebuah file .xlsx menjadi .txt dan membuka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .txt, yang merupakan format teks biasa (plain text). Jika kita simpan sebuah file .xlsx menjadi .csv dan membuka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .csv, yang merupakan format comma separated values. Jika kita simpan sebuah file .xlsx menjadi .pdf dan membuka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .pdf.Jika kita simpan sebuah file .xlsx menjadi .docx, yang terjadi adalah file .xlsx akan disimpan dengan ekstensi .docx, tetapi tidak akan dikonversi ke format .docx.Pelajari lebih lanjutMateri tentang beberapa aspek ilmu komputer https://brainly.co.id/tugas/16652646Materi tentang jenis software https://brainly.co.id/tugas/2300113Materi tentang beberapa software komputer https://brainly.co.id/tugas/6674111Detail Jawaban

Jawaban :

Pembahasan

ketika kita akan menyimpan suatu file pada software Microsoft, kita akan mendapatkan beberapa pilihan format.

.xlsx adalah format file Microsoft excel workbook pada tahun 2007, 2010 dan 2013 (2003 keatas). Excel berisi kolom dan baris, cara mengisinya seperti sistem koordinat. Excel biasanya digunakan untuk mengurutkan, membuat, meringkas, mengedit, menganalisa, data. Melakukan perhitungan statistika dan aritmatika.

.docx adalah format file Microsoft word untuk mengolah kata. Word berguna untuk word processing yaitu mengedit, membuat, dan memformat dokumen.

File bukan lah sebuah format, ia merupakan sebuah berkas komputer yang berisi sesuatu. File adalah suatu kumpulan informasi dan data yang saling terhubung dan disimpan dengan menggunakan format. Dalam Kamus Besar Bahasa Indonesia (KBBI) file adalah berkas yang merupakan suatu kesatuan arsip yang mengandung masalah tertentu dan disusun berdasarkan pola klasifikasi. (Sumber : Kepmendagri Nomor 30 Tahun 1979 tentang Tata Kearsipan Departemen Dalam Negeri).

win merupakan format sistem (tidak bisa dibuka secara sengaja kecuali dengan software pendukung). Gunanya adalah jika mengalami terhapusnya file penting, sistem dapat memulihkan.
