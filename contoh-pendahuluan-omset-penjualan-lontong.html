29. seorang karyawan menjual 50 bungkus lontong sayur yang menjual habis dengan harga rp325.000 jika dari hasil penjualan itu dia memperoleh keuntungan rp2.000 per bungkus berapakah harga pembelian ke seluruh lontong sayur tersebut?​


Penjelasan dengan langkah-langkah:

banyak lontong terjual = 50 bungkus

jumlah harga lontong terjual = Rp325.000,00

besar untung = Rp2.000,00/bungkus

-----------------

harga jual lontong per bungkus

= jumlah harga lontong terjual : banyak lontong terjual

= Rp325.000,00 : 50

= Rp6.500,00/bungkus

harga beli lontong per bungkus

= harga jual lontong per bungkus - besar untung per bungkus

= Rp6.500,00/bungkus - Rp2.000,00/bungkus

= Rp4.500,00/bungkus

------------------

jumlah harga beli keseluruhan

= harga beli lontong per bungkus x banyak lontong terjual

= Rp4.500,00 x 50

= Rp225.000,00
