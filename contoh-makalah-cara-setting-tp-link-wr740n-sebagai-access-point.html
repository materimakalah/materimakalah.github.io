2. langkah konfigurasi Access Point tp-link​


Telah terdapat langkah-langkah dalam konfigurasi Access Point TP-LINK yaitu ialah sebagai berikut:

Siapkanlah terlebih dahulu Access Point, kabel LAN, dan juga power dan setelah itu koneksikanlah kabel LAN pada komputer. Bukalah browser seperti Firefox, lalu ketikkanlah IP Address Access Point pada address bar lalu tekan enter.Masukkanlah username dan juga password dari Access Point. Kemudian, akan masuk ke halaman Access Point. Pada tempat ini telah digunakan untuk me-manage ataupun mengkonfigurasi segala macam pengaturan dan juga fungsi dari Access Point kemudian klik Next. Selanjutnya, pilihlah mode Access Point dan klik Next. Pada bagian selanjutnya yaitu ialah tempat untuk mengubah SSID dan juga mengisi password. Pada bagian network setting tidak perlu ada yang dirubah. Setelah itu, akan muncul halaman pemberitahuan maka dari itu pastikanlah semuanya telah benar dan pilih reboot. Kemudian, tunggulah hingga proses reboot telah selesai. Setelah proses reboot telah selesai maka dari itu kita dapat melihat laptop ataupun handphone untuk dapat mengetahui apakah wifinya telah berhasil dibuat. Apabila SSID yang telah diciptakan tersebut telah muncul maka dari itu proses konfigurasinya telah berhasil.Pembahasan

Pengertian dari TP-LINK yaitu ialah suatu perusahaan penyedia dunia untuk perangkat jaringan dan juga aksesoris yang dinilai terpercaya untuk dapat memenuhi segala macam aspek di dalam kehidupan kita sehari-hari.

32. Ada TP-LINK tl-wr840n.1. Sebutkan cara setting IP Network!2. Sebutkan cara setting IP LAN!3. Sebutkan cara setting routing!​


1. Cara setting IP Network pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router dengan mengetik alamat IP router (biasanya 192.168.0.1 atau 192.168.1.1) di URL.Masukkan username dan password default yang tercantum pada manual atau label pada router.Pilih "Network" atau "LAN" dari menu utama.Ubah pengaturan IP Network sesuai kebutuhan, misalnya dengan mengubah alamat IP, subnet mask, dan gateway.Klik "Save" atau "Apply" untuk menyimpan perubahan.

2. Cara setting IP LAN pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "LAN" dari menu utama.Ubah pengaturan IP LAN sesuai kebutuhan, misalnya dengan mengubah alamat IP dan subnet mask.Klik "Save" atau "Apply" untuk menyimpan perubahan.

3. Cara setting routing pada TP-LINK TL-WR840N:

Buka browser web dan masuk ke halaman admin router.Masukkan username dan password default.Pilih "Routing" dari menu utama.Pilih tipe routing yang dibutuhkan, seperti Static Routing atau Dynamic Routing.Masukkan informasi yang dibutuhkan untuk setiap tipe routing, misalnya destination IP, subnet mask, gateway, dan metric.Klik "Save" atau "Apply" untuk menyimpan perubahan.

note: Langkah-langkah ini mungkin berbeda tergantung dari versi firmware router. Pastikan untuk membaca manual pengguna atau dokumentasi produk untuk informasi lebih rinci.

38. Bagaimana cara access point menciptakan jaringan WLAN


Penggunaan acces point pada jaringan WLAN adalah dengan melakukan pengaturan terlebih dahulu pada sebuah acces point. Untuk melakukan pengaturan, kita bisa menggunakan langkah sebagai berikut

Masuk ke control panelBerikan IP pada perangkat laptop kitaAkses IP default pada access point kita, biasanya tertulis pada bagian bawah acces pointLakukan pengaturan lebih lanjut pada acces point

Setelah access point memiliki alamat IP, kita bisa memberikan alamat IP pada perangkat-perangkat laptop atau komputer kita. Apabila IP bersifat static, maka kita harus memberikan IP satu per satu. Namun kita bisa melakukan pengaturan dengan merubahnya menjadi IP dynamic, perangkat akan mendapatkan IP secara otomatis dari perangkat access point.

Pembahasan

Setelah access point sudah terhubung dengan perangkat kita, kita bisa melakukan test dengan cara melakukan ping. Beberapa kemudahan setelah berhasil membuat jaringan WLAN adalah

Berbagi dataBerbagi printerBerbagi koneksi internetPelajari lebih lanjut

Demikian pembahasan mengenai cara access point menciptakan jaringan WLAN, access point berfungsi sebagai penghubung perangkat yang satu dengan perangkat lainnya. Untuk mempelajari lebih lanjut mengenai jaringan WLAN dapat dibaca pada link berikut
