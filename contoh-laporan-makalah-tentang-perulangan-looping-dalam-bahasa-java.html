16. Apabila kita ingin melakukan sebuah perulangan(looping) maka fungsi yang harus digunakan adalah...​


Jawaban singkat:

Loop while dan Loop for

Penjelasan:

Pada bahasa pemograman Java, jika ingin melakukan looping bisa menggunakan 4 cara sesuai kondisi yang ada:

1. Looping while  

Sintaks :

while (condition) {  

   //argument

}

Jika ingin menggunakan while, maka harus membuat sebuah variabel counter agar condition bisa terpenuhi sehingga looping punya titik berhenti. Misalnya sebuah variabel i yang awalnya bernilai 0, lalu looping dengan kondisi i < 5, seperti:

int i = 0;

while (i < 5) {  

   //argument

   i++;

}

Nilai i awalnya 0, dengan increment i++ maka sampai i mencapai nilai 5 baru looping akan berhenti.

2. Looping do while

Sintaks:

do {

  //argument

} while (condition)

Konsepnya sama seperti whlie, tetapi yang membedakan, argumen/perintah dijalankan terlebih dahulu baru kondisi dicek. Pada looping while, program akan melakukan pengecekan terlebih dahulu apakah memenuhi kondisi, jika memenuhi maka jalankan argument.

contoh:

int i = 0;

do {

  //argument

  i++;

} while (i < 5)  

3. Looping for

Sintaks:

for (initialization; condition; update) {

   // argument

}

Looping for adalah salah satu cara yang umum digunakan untuk melakukan looping. Strukturnya terdiri dari tiga bagian: initialization, condition, dan update.

- initialization: Pada bagian ini, biasanya digunakan untuk menginisialisasi variabel penghitung atau variabel lain yang digunakan dalam loop.

- condition: Merupakan kondisi yang akan diperiksa pada setiap iterasi. Jika kondisi bernilai true, maka loop akan terus berlanjut; jika bernilai false, loop akan berhenti.

- update: Pada bagian ini, biasanya digunakan untuk mengubah nilai variabel penghitung atau variabel lain yang akan mempengaruhi kondisi loop.

Contoh penggunaan looping for:

for (int i = 0; i < 5; i++) {

   // argument

}

Pada contoh di atas, loop akan dijalankan selama i kurang dari 5, dan setiap iterasi, nilai i akan bertambah satu (dikarenakan i++ di bagian update).

4. Looping foreach

Sintaks:

for (Type var : iterable) {

   // argument

}

Looping foreach digunakan untuk looping elemen-elemen dari suatu koleksi objek (seperti array atau koleksi lainnya) secara lebih mudah dibandingkan dengan loop for konvensional.

- Type: Tipe data dari elemen-elemen dalam koleksi.

- var: Nama variabel yang akan digunakan untuk menyimpan setiap elemen dalam iterasi.

- iterable: Objek koleksi yang akan diiterasi.

Contoh penggunaan looping foreach untuk mengiterasi array:

int[] numbers = {1, 2, 3, 4, 5};

for (int num : numbers) {

   // argument

}

Pada contoh di atas, setiap elemen dalam array numbers akan disimpan dalam variabel num secara berurutan pada setiap iterasi, sehingga program dapat melakukan operasi pada setiap elemen tersebut.


17. Buatlah program java looping untuk soal dibawah ini


Program dalam bahasa java

// AUTHOR : LordRimuruTempest

public class main {

   public static void main(String[] args) {

       int temp = 1;

       for (int i = 1; i <= 15; i++) {

           System.out.print(i + " ");

           if(i == ((temp + 1) * temp) / 2){

               System.out.println();

               temp += 1;

           }

       }

   }

}

Pembahasan

Untuk membuat output seperti pada soal, perlu diketahui dulu cara kerja dari program tersebut. Perhatikan bahwa bilangan terakhir pada tiap baris memiliki pola sebagai berikut

1 = 1

3 = 1 + 2

6 = 1 + 2 + 3

10 = 1 + 2 + 3 + 4

15 = 1 + 2 + 3 + 4 + 5

atau dengan kata lain, bilangan terakhir pada baris ke-n adalah

1 + 2 + 3 + ... + n = ((n + 1) x n) / 2

Sehingga alur program diatas dapat ditulis sebagai berikut :

1. Program dimulai

2. Buat variabel temp untuk menyimpan nomor baris

3. Beri nilai awal temp = 1

2. lakukan perulangan dari indeks i = 1 sampai i = 15

cetak i tanpa membuat baris barujika i = ((temp + 1) x temp) / 2 maka ganti baris, dan nomor baris bertambah 1 yaitu temp = temp + 1

5. Program selesai

Alur/algoritma program dapat ditulis dalam pseudocode sebagai berikut

Algoritma program

program start

var temp : integer

temp <-- 1

for i = 1 to i = 15 do

   print i  //no new line

   if(i = ((temp + 1) * temp) / 2) then

       //new line

       temp <-- temp + 1

   end if

end for

program end

Untuk menerapkan algoritma diatas ke dalam program java, pertama-tama buat class java(nama kelas bebas) beserta main method-nya. Deklarasikan variabel temp dengan tipe integer dan beri nilai awal temp = 1. Kemudian lakukan perulangan(looping) menggunakan for sesuai dengan arahan dari algoritma/alur program. 19. mengapa pada nested loop FOR- NEXT (perulangan di dalam perulangan) looping yang didalam akan lebih banyak dilakukan perulangan dibandingkan yang di luar ?


Nested Loop atau perulangan bersarang adalah sebuah perulangan yang terdapat di dalam perulangan. Bentuk nested loop for sendiri kurang lebih adalah seperti berikut

For i = 1 To 5

         For j = 1 To 5        


         Next

Next

For-Next di dalam akan memiliki lebih banyak perulangan karena perulangan di dalam akan diulang sebanyak jumlah perulangan yang ada di luar.

Sebagai contoh potongan kode di atas For-Next di luar hanya diulang sebanyak 5 kali sedangkan, For-Next di dalam akan diulang setiap kali perulangan di luar dilakukan sehingga terjadi 25 kali perulangan.

Jadi pada saat i = 1 terjadi 5 kali perulangan di For-Next dalam, kemudian i =2 terjadi 5 kali perulangan, begitu seterusnya hingga terjadi 5x5 perulangan untuk perulangan For-Next yang di dalam.

31. algoritma yang menjalankan beberapa langkah tertentu secara berulang-ulang atau looping disebut​


Algoritma yang menjalankan beberapa langkah tertentu secara berulang-ulang atau looping disebut "algoritma pengulangan" atau "algoritma iterasi." Ini adalah salah satu konsep dasar dalam pemrograman yang memungkinkan komputer atau perangkat lunak untuk menjalankan serangkaian perintah atau instruksi berulang kali selama kondisi tertentu terpenuhi. Pengulangan digunakan untuk mengotomatisasi tugas-tugas yang perlu diulang, seperti pemrosesan data dalam jumlah besar atau melakukan operasi berulang dalam perulangan waktu. Ada berbagai jenis pengulangan, termasuk pengulangan while, for, dan do-while, yang digunakan dalam berbagai bahasa pemrograman.

38. contoh pengulangaan ( looping ) pada organisasi


Jawaban:

Perulangan (Lopping)

1. Perulangan while

While merupakan instruksi perulangan yang mirip dengan perulangan for. Bentuk perulangan while dikendalikan oleh syarat tertentu , yaitu perulangan akan terus dilaksanakan selama syarat tersebut terpenuhi.

Bentuk umum pernyataan While adalah:

while(syarat)

{

Pernyataan/perintah;

}

Berikut contoh program sederhana menampilkan bilangan ganjil antar 1-10 dengan menggunakan While:

//Program Sederhana Dengan While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

while(nilai<=10)

{

cout<<nilai<<" ";

nilai+=2;

}

getch();

}

Hasil outputnya nanti seperti ini

1 3 5 7 9

2. Pernyataan Do....While...

- Pada perulangan Do-While, tubuh loop berupa pernyataan.

- Pada pernyataan Do, mula-mula pernyataan dijalankan.

- Selanjutnya, kondisi diuji. Seandainya kondisi bernilai benar, maka pernyataan dijalankan lagi. Kemudian kondisi diperiksa kembali dan seterusnya.

- Kalau kondisii bernilai salah, maka pernyataan tidak dijalankan lagi

- Pada dasarnya struktur perulangan Do...While sama saja dengan struktur while, hanya saja pada proses perulangan dengan while, seleksi berada di while yang letaknya berada di atas sementara pada perulangan Do...While seleksi berada dibawah batas perulangan. Jadi dengan menggunakan Do...While sekurang-kurangnya akan terjadi satu kali perulangan.

Bentuk umum Do-While

do

{

Perintah/Pernyataan;

}

While(syarat);

Berikut contoh program sederhana menampilkan bilangan ganjil antara 1-10 dengan Do While :

//Program Sederhana Dengan Perintah Do While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

do

{

cout<<nilai<<" ";

nilai+=2;

}

while(nilai<=10);

getch();

}


39. apakah mungkin untuk membuat suatu perulangan tanpa henti (infinite looping)? struktur perulangan mana yang dapat digunakan? jelaskan!


Mungkin, dengan membuat syarat terjadinya looping terpenuhi terus (tidak ada perubahan pada variabel yang dibuat syarat). Struktur while dan for dapat digunakan untuk infinite looping. Contoh :

for (a=1; a<5; a--) {

cout<<"Hello";

}

Kode C++ diatas akan menampilkan kata "Hello" berulang - ulang tanpa henti karena syarat pengulangannya terpenuhi terus, yaitu a<5, sedangkan a = 1. Dan perubahannya bukannlah a++, namun a-- yang membuat nilai a semakin berkurang.
