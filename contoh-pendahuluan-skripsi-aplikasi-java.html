* `Math.pow()` adalah operator pangkat

* `Math.round()` adalah operator aritmatika bulat

java

double area = Math.PI * Math.pow(r, 2);

System.out.println("Area lingkaran = " + area);

example

Sure! Here's an example using code:

java

double r1 = 2.0; // Radius

double area = Math.PI * Math.pow(r1, 2);

System.out.println("Area is " + area);

This code outputs:

Area is 4pi

This calculates the area of a circle with radius r1, using the formula:

$$Area = \pi r^2$$
