12. sebuah kapal berlayar ke arah utara sejauh 12.000 m. dan kemudian belok kearah timur sejauh 16.000 m, msks jarak kapal terhadap titik awal


Mapel : Matematika
Kelas : VIII SMP
Bab : Teorema Pythagoras

Pembahasan :
Skema Kapal berlayar membentuk Segitiga siku siku, dengan :

Alas = 12.000 m
Tinggi = 16.000 m

Jarak Kapal terhadap titik awal = Sisi Miring pada Segitiga
= √(Alas² + Tinggi²)
= √(12.000² + 16.000²)
= √400.000.000
= √(4 x 10^8)
= 2 x 10^4
= 20.000 mMenggunakan rumus pythagoraa

ubah satuan meter ke km untuk memudahkan

12.000 m = 12 km
16.000 m = 16 km

jarak kapal terhadap titik awal

=√16²+12²
=√256+144
=√400
= 20 km

jadi jaraknya adalah 20km atau 20.000 m

21. seorang manajer perusahaan furniture menganalisa bahwa biaya prpdukdi 1 set kursi tamu bergantung linier terhadap banysknya set kursi yg diproduksi, jika memproduksi 100 set msks biaya produksi per setnys adalah 2.200 jika memprodukdi 300 maka biaya produkdinya 4.800 tenukan persamaan linier harga y terhadsp banysk kursi yg diprodukdi x, gambarlsh grsfiknya . berspa biaya produksi 75 set kursi . tentukan kemitingsn grsfik tersebut apakah yg dipresentasikan kemiringan tersebut. tentukan titik potong grafik terhadap sumbu y aps yg dipresentasikan titik potong tersebut


1. Misalkan:

y = biaya produksi kursi tamu per set

x = banyaknya kursi tamu (dalam set)

y bergantung linear terhadap x, maka dapat dituliskan:

y = f(x), dengan f(x) sebuah fungsi linear.

Misalkan pula f(x) = ax + b untuk suatu a, b ∈ bilangan real dan a ≠ 0,

maka y = ax + b

Kita tentukan nilai a dan b dari data yang diberikan.

Untuk x = 100 maka y = 2.200.000

artinya, 2.200.000 = 100a + b

Untuk x = 300 maka y = 4.800.000

Artinya, 4.800.000 = 300a + b

Eliminasi b dari kedua persamaan.

2.200.000 = 100a + b

4.800.000 = 300a + b

-----------------------------  -

-2.600.000 = -200a

↔ 200a = 2.600.000

↔ a = 13.000

Akibatnya:

2.200.000 = 100a + b

↔ 2.200.000 = 100(13.000) + b

↔ 2.200.000 = 1.300.000 + b

↔ 2.200.000 – 1.300.000 = b

↔ b = 900.000

Jadi, persamaan linear yang memodelkan kasus di atas adalah

y = 13.000x + 900.000
{untuk grafik silakan gambarkan sendiri sebagai latihan ya.}

2. Untuk x = 75, maka
y = 13.000(75) + 900.000
   = 975.000 + 900.000
   = 1.875.000
Jadi, biaya produksi untuk 75 set kursi adalah Rp 1.875.000

3. y = 13.000x + 900.000
Kemiringan garis tersebut adalah 13.000.
Angka 13.000 merupakan koefisien terikat pada variabel x, artinya angka 13.000 merupakan biaya produksi yang diakibatkan/muncul dari setiap set kursi yang diproduksi. Dalam ilmu ekonomi, biaya seperti ini dinamakan biaya variabel.
 
4. Titik potong grafik terhadap sumbu y adalah ketika x = 0.
y = 13.000(0) + 900.000
   = 900.000
Jadi, titik potong terhadap sumbu y adalah (0,900.000)
Angka 900.000 merupakan koefisien dari persamaan linear. Angka ini tidak akan berubah, berapa pun nilai dari x. Biasanya biaya seperti ini dinamakan biaya tetap.
